- Export `async function getAudioContext(): Promise<AudioContext>`
- Maintains a module-level `let _ctx: AudioContext | null = null`

Create `src/engine/sounds/singingBowl.ts`:
- Internal helper `function createPartial(ac: AudioContext, freq: number, peakGain: number, attackTime: number, decayTime: number, startTime: number): void`
  - Creates `new OscillatorNode(ac, { type: 'sine', frequency: freq })`
  - Creates `new GainNode(ac, { gain: 0 })`
  - Schedules envelope: `gainNode.gain.setValueAtTime(0, startTime)` then `gainNode.gain.linearRampToValueAtTime(peakGain, startTime + attackTime)` then `gainNode.gain.exponentialRampToValueAtTime(0.001, startTime + attackTime + decayTime)`
  - Connects: `osc.connect(gainNode).connect(ac.destination)`
  - Starts and stops: `osc.start(startTime)`, `osc.stop(startTime + attackTime + decayTime + 0.1)`
- Export `function strikeBowl(ac: AudioContext, masterGain: number = 1.0): void`
  - Uses `ac.currentTime` as `now`
  - Fundamental = 220 Hz (per D-01: warm and deep, low fundamental)
  - Creates 5 partials with these exact parameters (per D-01 and D-02):
    1. `createPartial(ac, 220, 0.35 * masterGain, 0.08, 6.0, now)` — fundamental
    2. `createPartial(ac, 220 * 2.76, 0.20 * masterGain, 0.05, 4.5, now)` — 1st inharmonic partial
    3. `createPartial(ac, 220 * 4.72, 0.12 * masterGain, 0.03, 3.5, now)` — 2nd inharmonic partial
    4. `createPartial(ac, 220 * 6.83, 0.06 * masterGain, 0.02, 2.5, now)` — 3rd inharmonic partial
    5. `createPartial(ac, 220 * 1.003, 0.15 * masterGain, 0.08, 5.5, now)` — detuned shimmer (per D-02)

Create `src/engine/sounds/phase3Tone.ts` (per D-05: contrasting urgent tone, Claude's discretion — rising sine swell):
- Export `function createPhase3Ramp(ac: AudioContext, durationSec: number): GainNode`
  - Creates `new GainNode(ac, { gain: 0 })`
  - Anchors: `masterGain.gain.setValueAtTime(0, ac.currentTime)` (CRITICAL: anchor before ramp per research pitfall 6)
  - Ramps: `masterGain.gain.linearRampToValueAtTime(1.0, ac.currentTime + durationSec)` (per AUD-02: 0% to 100%)
  - Connects to `ac.destination`
  - Returns the GainNode (so AlarmEngine can connect sound sources through it)
- Export `function startPhase3Swell(ac: AudioContext, masterGain: GainNode): OscillatorNode`
  - Creates `new OscillatorNode(ac, { type: 'sine', frequency: 80 })`
  - Creates local `new GainNode(ac, { gain: 0 })` for soft attack
  - Schedules frequency sweep: `osc.frequency.setValueAtTime(80, ac.currentTime)` then `osc.frequency.linearRampToValueAtTime(220, ac.currentTime + 3.0)`
  - Schedules soft attack on local gain: `oscGain.gain.setValueAtTime(0, ac.currentTime)` then `oscGain.gain.linearRampToValueAtTime(1.0, ac.currentTime + 0.5)`
  - Connects: `osc.connect(oscGain).connect(masterGain)` (NOT directly to destination — goes through the ramp GainNode)
  - `osc.start()`
  - Returns the OscillatorNode (caller stops it after ~3s and creates a new one for looping)
- Export `function fadeOutGain(gain: GainNode, ac: AudioContext): void`
  - `gain.gain.setTargetAtTime(0.0001, ac.currentTime, 0.015)` (smooth fade, no click per research pitfall 4)
